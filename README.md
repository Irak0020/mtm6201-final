# mtm6201-final

This is my final website for MTM6201. I created this project using Figma designs from my UX course. The website features three pages: **Home**, **About**, and **Contact**. 

## Technologies Used
- **Bootstrap** - responsive layout
- **CSS variables** - custom colour customization
- **Google Fonts** - typography
- **Animate.css** - required public CSS library

## Accessibility Features
- Alt text for images
- Semantic HTML
- Skip link navigation
- Clear form labels

## Challenges & Solutions
- **Image paths**: Fixed file names containing spaces
- **Spacing**: Used Bootstrap classes and custom CSS to match Figma design

## Key Learnings
- Converting Figma wireframes to code
- Bootstrap implementation
- Web accessibility best practices
- GitHub Pages deployment
